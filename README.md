## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/SyedaRidaSalam/syeda-rida-salam.git
```

### Navigate to the Project

```bash
cd syeda-rida-salam
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:3000
```
